## [0.1.0] - 2025-02-13

### Added
- Initial release of Oxide theme for VS Code
- Complete syntax highlighting for all major languages with semantic color assignments:
  - Functions in red for clear identification
  - Strings in green for data representation
  - Keywords in purple for control flow
  - Variables in blue for identifiers
  - Types/Classes in sky blue for structure definitions
  - Constants/Numbers in orange for literals
  - Comments in subtle gray for de-emphasis
- Comprehensive UI theming with minimalist gray palette
- Full terminal color integration with ANSI color support
- Git decoration colors for version control status
- Semantic highlighting support for enhanced token coloring
- Symbol icon theming with consistent color application
- Debug mode contextual highlighting
- Bracket pair colorization support
- Diff editor highlighting with semantic colors
- Search and selection highlighting
- Peek view and widget theming
- Status bar, activity bar, and sidebar theming

### Philosophy
- Function-first color application: colors convey information
- Visual silence through intentional restraint
- Systematic harmony using OKLCH color space
- Whites and grays as foundation, colors for semantic meaning
