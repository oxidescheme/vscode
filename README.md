## Installation

### Via VS Code Marketplace

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for "Oxide Theme"
4. Click Install
5. Press `Ctrl+Shift+P` / `Cmd+Shift+P`
6. Type "Preferences: Color Theme"
7. Select "Oxide"

### Via Command Line

```bash
code --install-extension oxidescheme.oxide-theme
```
